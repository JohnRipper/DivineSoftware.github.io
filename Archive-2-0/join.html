Preparation

You will need a socket client because it seemed too lame to me to allow joining just everyone, you will have to make own tcp client to talk with us. Not too hard, not too easy like captcha, dont even try to use putty, I'll make sure it won't be compatible with it.

Server acts like this:
1. Client connects, server responds with 'Enter your username:'
2. Client sends a message with the username
3. Server announces the new client
4. Client sends another message
5. Server sends message to all clients (including yours)
So basically an A and B (message and answer) example.

Communication uses UTF-8 encoding!
Current server is written in Python (Ruby version is in developement), but you are allowed to write your client in whatever language you want to.
Server ip is on 193.161.193.99 and port 33036
